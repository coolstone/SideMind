首次安装建议

先恢复已有备份，或完成配置后导出一份本地备份。普通更新不会清空数据，但加载另一个解压目录可能产生新的扩展身份。

备份内容

模型配置、响应语言、启动偏好、提示词、聊天空间、聊天历史和本地工件。备份文件只保存在你选择的本地目录，不会上传。

盾
独立开发版的安全边界

API Key 保存在 chrome.storage.local，并限制为扩展受信任页面可读。个人本机使用足够方便；若要发布给用户，请增加服务端代理、用户鉴权、配额与审计。

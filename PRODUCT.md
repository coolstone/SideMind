# SideMind 产品说明

## 产品判断

Sider 的核心价值不是“把聊天框放到侧边”，而是把浏览器正在发生的事情变成 AI 可用的上下文，并让 AI 结果能回到浏览器工作流。SideMind 首版因此优先完成一个闭环：

```text
页面 / 选区 / 截图 → API 直连或 ChatGPT 网页交接 → 复制 / 写回网页 / 连续追问
```

## 首版用户任务

1. 阅读：总结长文、解释概念、提取关键数据。
2. 语言：划词翻译、跨语言问答、内容改写。
3. 写作：在邮件、评论、表单等输入框旁生成并写回文本。
4. 视觉：网页结构难以提取时，让模型直接理解当前可见截图。
5. 连续协作：围绕同一页面追问，并能回看历史对话。

## 与成熟 Sider 的差距

SideMind 目前覆盖的是“上下文 Chat”核心，不假装已经具备成熟产品的全部能力。Sider 公开能力还包含多模型群聊、PDF / YouTube 专用理解、跨站浏览器 Agent，以及用自然语言持久化修改网站。后两类能力需要权限治理、可回滚执行、敏感操作确认和站点适配，不适合未经安全设计就直接塞进首版。

## V0.2 双连接模式

- API Key：OpenAI、DeepSeek、OpenRouter 与自定义 OpenAI 兼容接口，结果直接流回 SideMind。
- ChatGPT 网页交接：利用用户正常登录的网页订阅，把提示词写入可见输入框，但始终由用户手动发送。
- 回答同步：只解析 ChatGPT 页面上可见的最新助手消息，并由用户主动同步回侧边栏。
- 硬边界：不读取或导出 Cookie，不调用逆向私有接口，不自动发送，不拦截响应流量。

## V0.3 工作台能力

- 提示词库：维护、分类、搜索、显示/隐藏、拖动或按钮排序，并在侧栏按自定义顺序直接执行。
- 模型配置：把服务商、模型、接口、Key 和输出预算组成独立档案，在对话中快速切换。
- 上下文入口：网页正文、本地 Markdown/TXT/HTML 等文本文件、选区、截图、上传或剪贴板粘贴的图片附件，以及文本附件统一进入请求构建流程。
- 聊天空间：不同项目的历史隔离保存，支持空间内全文搜索、自定义聊天标题，以及当前空间 Markdown 的导入、去重合并和导出；自定义名称不会被后续消息覆盖。启动时可恢复上次会话或始终新建，并在历史列表中高亮定位当前会话。
- 本地备份：设置页可把模型配置、提示词、空间、历史、偏好和工件导出为本地 JSON 并恢复；API Key 默认排除，只有用户明确确认后才写入备份。首次安装主动提示建立备份。
- 自愿支持：顶部爱心按钮打开本地捐赠弹窗，支持微信支付与支付宝二维码切换；不联网、不采集支付信息，也不影响任何功能权限。
- 思考控制：OpenAI Responses 使用 `reasoning.effort`；DeepSeek 使用 `thinking.type` 和 `reasoning_effort`；网页交接模式转换为明确的思考提示。
- 稳定性预算：附件单文件 3MB、单次总量 5MB；网页、历史消息和文本附件采用独立字符预算，避免超大请求拖慢浏览器或超过模型上下文。
- 回答再加工：每条回答支持复制、保存为本地 Markdown、引用、重新生成、分享、朗读、写回网页和确认删除；删除时同步清理历史消息与关联工件缓存。
- 提示词迁移：根据 Sider 提示词列表建立等效模板库，通过版本号自动补充新模板，按 ID/名称去重并保留用户已有内容。
- 模板变量：`${lang}` 自动读取响应语言设置，`${input}` 优先使用网页选区或输入框，其他变量在运行前动态收集。
- 提示词运行入口：输入框上方展示常用提示词，溢出菜单提供完整列表，并显式展示当前网页选区来源。
- 高频交互保护：网页选区实时同步；新聊天提供显式入口，草稿清空前确认，生成过程中禁止切换或清空历史。
- 本地工件：从模型回答中识别网页与代码文件，生成可点击工件卡片；独立查看器支持代码、预览、分栏、本地编辑、复制和下载。
- 工件安全边界：内容存入受限的 `chrome.storage.local`；预览 iframe 不授予沙箱权限，并用 CSP 禁止脚本、表单提交和外部网络资源。

## 推荐路线

### V0.4：阅读质量

- Mozilla Readability 正文提取
- PDF.js 文本层与页码引用
- YouTube 字幕与时间戳引用
- 流式输出、停止生成、重新生成
- 回答中的证据片段定位与页面高亮

### V0.5：写作与知识

- 可组合 Prompt / Skill
- 常用语气、品牌术语表与个人偏好
- 文件上传与多页面知识集合
- 对话导出 Markdown / Notion / Google Docs

### V0.6：可控浏览器 Agent

- 只读任务：搜索、比较、提取、整理
- 写操作前预览计划并逐步确认
- 站点级权限、域名白名单、操作记录与撤销
- 密码、支付、删除、发布等高风险动作硬性拦截

### 商业化前置

- 服务端模型网关与多租户账户
- 套餐、配额、成本与延迟路由
- 隐私政策、数据保留策略、企业审计
- Prompt 注入防御评测与浏览器权限最小化复核
